# Habit Tracker

A single-file, WHOOP-styled habit tracker web app. Tracks workouts (binary) and Leetcode questions (count) on a GitHub-style contribution heatmap, with weekly stats, streaks, and an 8-week trend.

Built with React 18 (via CDN) + Babel standalone — no build step. State persists to `localStorage`.

## Run locally

Just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```

## Deploy

Static site — deploys to any static host. Live on Vercel.
